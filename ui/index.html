Enigma Machine in JS

History buffs and movie buffs(I'm looking at you, The Imitation Game) alike know about the wonders that encryption did for the war effort. I tried to emulate the Enigma machine, which helped British intelligence intercept many German codes, using Javascript/Jquery. So far, there are three rotors, unlike the actual Enigma that had 5 or 7.
The plugboard and lightboard are functional. The rows in the middle are the buttons that add a letter to the input plaintext. The encrypted letter glows in the lightboard which is the upper set of rows. The lower set of rows represents the plugboard. The first and second letter to be clicked are swapped in the code, before they go through any rotors. This adds an added layer of encryption.
